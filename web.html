What's changed
Starting in iOS 13.4, download links for all wallpapers in this app will open in an in-app preview instead of Safari due to a change Apple made.

Because of this, there is now a different method to save wallpapers to your photos:
1. Tap on ↓ on the wallpaper you'd like to download.
2. Tap on the share button.
3. Tap on Save image.
Got it!
How to install?
1. Open this webpage in Safari on your iPhone or iPad.
2. Tap on the share button.
3. Tap on Add to Home Screen.
4. Tap on Add.

Updates.

Various changes to the way our app works in the background. Load times can be about 15% faster and the app is more future-proof.
Version 2.1.2
This fixes the download button of the wallpaper from January 15, 2020. It now works as intended.
Version 2.1.1
We've introduced instructions for iOS 13.4 and newer as the process of downloading an image has changed.
Version 2.1
In this version we have improved the performance of the animation for opening the + menu. It should be much smoother now.
Version 2.0.3
Fixes a bug that stopped wallpapers from fading out when getting to the bottom of the screen by scrolling.
Version 2.0.2
The animation for opening the + menu on the start screen has been fixed.
Version 2.0.1
When using your iPhone in landscape and light mode, there was a black gradient next to the wallpapers. We’ve fixed this bug in this update.
Version 2.0
With this update we are introducing collections and a whole new look for the home screen. This version also improves compatibility with iOS 13.1 and iPadOS 13.1.
Version 1.1
This update adds pull-to-refresh functionality to the homescreen and contains this new about page.
Version 1.0.1
This version fixes a bug that could cause the wallpapers on the home screen to be invisible on devices running older versions of iOS.
Version 1.0
